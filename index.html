Named Entity Recognition (NER) merupakan bidang NLP yang dapat mengenali jenis entitas pada kata dalam sebuah kalimat.

NER juga dapat digunakan untuk bidang NLP lainnya seperti Question Answering System, Chatbot, dan sebagainya.

Algoritma yang biasanya digunakan adalah Algoritma Rule-Based, Machine Learning (Supervised atau Unsupervised) dan Hybrid.

Salah satu algoritma state-of-the-art pada bidang ini adalah Algoritma Machine Learning Unsupervised, seperti Bidirectional Long Short Term Memory (BiLSTM), Conditional Random Field (CRF), dan lain sebagainya
